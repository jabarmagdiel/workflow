Exportar Backup

Descarga un archivo .json con todos los workflows, instancias, tareas y eventos de auditoría del sistema. Úsalo para respaldo o migración.

  • ✓ Incluye workflows (DRAFT y PUBLISHED)
  • ✓ Instancias y su historial completo
  • ✓ Todas las tareas
  • ✓ Eventos de bitácora
📥

Restaurar Backup

Sube un archivo de backup generado por BPFlow para restaurar datos. Solo se importan las entidades que no existan en el sistema actual (sin sobrescribir).

  • ⚠ Solo acepta archivos .json de BPFlow
  • ⚠ No sobreescribe registros existentes
  • ⚠ Requiere rol ADMIN
💡

Se recomienda programar backups periódicos en entornos de producción. El archivo exportado es portable y puede importarse en cualquier instancia de BPFlow.
